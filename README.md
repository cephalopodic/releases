# Cephalopodic Releases

This repository hosts public binary release artifacts for Cephalopodic tools.

Release tags are tool-scoped to avoid collisions across binaries:

- `cephi-v0.2.0`
- `fortress-v0.1.3`
- `scath-v0.4.1`

Attached assets follow a uniform naming pattern:

- `<tool>-<version>-aarch64-apple-darwin.tar.gz`
- `<tool>-<version>-x86_64-apple-darwin.tar.gz`
- `<tool>-<version>-x86_64-unknown-linux-gnu.tar.gz`

Each archive also ships with a matching `.sha256` file.
